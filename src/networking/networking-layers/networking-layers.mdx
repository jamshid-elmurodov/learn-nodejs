# Networking layers
1. Physical Layer
    - Bitlarni uzatish (elektr signal, optik signal, radio to‘lqinlar orqali).
    - Kabel, Wi-Fi, Bluetooth kabi texnologiyalar shu yerda ishlaydi.

2. Data Link Layer
    - Bitta tarmoq (LAN) ichida aloqa o‘rnatish.
    - MAC manzillar orqali qurilmalarni aniqlash.
    - Frame (data blok) yaratadi va kimga yuborilishini boshqaradi.

3. Network Layer
    - Paketlarni manzildan manzilga yo‘naltirish (routing).
    - IP manzillar bilan ishlaydi.
    - Routerlar shu qatlamda ishlaydi.

4. Transport Layer (TCP/UDP)
    - Ikki qurilma orasida ishonchli yoki tez ma'lumot almashinuvi.
    - TCP – ishonchli, tartibli, qayta yuborish bilan (masalan: veb sahifalar).
    - UDP – tez, ammo ishonchsiz (masalan: video qo‘ng‘iroqlar, streamlar).

5. Session Layer
    - Aloqani ochish, boshqarish va yopish.
    - Masalan: login sessiyasi, audio/video chat sessiyasi.

6. Presentation Layer
    - Ma’lumot formatini tayyorlash:
       - Encode/Decode (masalan, JSON, XML).
       - Encrypt/Decrypt (masalan, SSL/TLS).
    - Turli tizimlar orasida ma’lumotni almashish.

7. Application Layer
    - Protokollar: HTTP, FTP, DNS, SMTP va boshqalar.
